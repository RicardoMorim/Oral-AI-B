O que é uma linguagem de programação

Uma linguagem de programação é o que um programador utiliza para dar ordens ao computador. Podendo ser utilizadas desde para automatizar tarefas difícieis até para o lazer. Os computadores junto com as linguagens mudaram a forma com que vivemos o mundo!

Ou seja, as linguagens de programação servem para se criar o que chamamos de software, ou seja, para "dar ordens" aos componentes físicos, ou hardware

No inicio apenas existia o código binário que como podemos ver nas tabelas ao lado os numeros podem ser representados de várias maneiras, na programação os mais utilizados são o binário, o decimal, e o hexadecimal

A tabela ASCII demonstra como o computador representa cada caractere. Para o computador, a letra "A" é 65 em decimal ou 41 em hexadecimal ou 1000001 em binario.

Para converter um número de binário para decimal é bastante fáci. Vamos converter o número 1000001, ou seja, a letra "A". Para fazer isso basta:

  • Começar da direita para a esquerda escolher um digito de cada vez e anotá-lo;
  • Se for um 1, fazer 2 elevado à ordem do digito -1 (por exemplo se for o primeiro digito fica 2^0 se for o segundo fica 2^1 etc.) e anotar o valor. Se for um 0 não se faz nada;
  • Repetir esse processo até ao fim do número
  • Somar todos os valores anotados.

Para o número 1000001 obteríamos 2^0+2^6 chegando assim ao resultado obtido 65.




Linguagem C

A linguagem C é uma linguagem de programação compilada de propósito geral. C é uma das linguagens de programação mais populares, mesmo tendo sido criada a muito tempo, e mesmo sendo difícil de se aprender. Um dos motivos para isto é a grande otimização desta linguagem, que compensa as dificuldades.

C é utilizado em várias áreas, como a criação de jogos, criação de sistemas operacionais, e qualquer tipo de aplicação que requira um rápido tempo de resposta.

Os maiores pontos fortes são

  • Rapidez;
  • Versatilidade;
  • Grande comunidade;
  • Muitas bibliotecas de código;

Os pontos negativos desta linguagem são todos à base da complexidade da mesma.




Python

Python é uma linguagem de programação de alto nível, interpretada , imperativa, orientada a objetos, funcional e de tipagem dinâmica.

Python está constantemente no topo da tabela das linguagens mais utilizadas, maioritariamente graças à facilidade de aprendizagem e a facilidade de compreensão. Ulizada maioritariamente para:

  • desenvolvimento Web;
  • Desenvimento de jogos;
  • Inteligência artificial e "Machine Learning";
  • Visualização de dados e manuseamento de dados.

Os pontos fortes desta linguagem são:

  • Facilidade de aprendizagem;
  • Facilidade de leitura de código;
  • Grande comunidade;
  • Versatilidade.

O maior ponto fraco desta linguagem é que por ser interpretada é mais lenta do as compiladas.

SQL

Structured Query Language, ou Linguagem de Consulta Estruturada ou SQL, é a linguagem de pesquisa declarativa padrão para banco de dados relacional (base de dados relacional).

Esta linguagem serve basicamente para armazenar e ler dados. Esta armazena os dados num arquivo em forma de tabelas e o utilziador pode especificar que coluna(s) ou linha(s) da tabela quer que sejam exibidas. No site que eu demonstrei ao falar de python utiliza esta linguagem para armazenar e ler os aniversários e os nomes das pessoas.

Javascript

JavaScript (frequentemente abreviado como JS) é uma linguagem de programação de alto nível, bastante fácil de aprender, porém bastante poderosa.

Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web. JavaScript permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos da web. A grande maioria dos sites usa, e todos os principais navegadores têm um mecanismo JavaScript dedicado para executá-lo.

Foi a primeira linguagem que eu aprendi, e é certamente uma linguagem que eu recomendo a qualquer um aprender, pois esta linguagem é bastante versátil, ganhando espaço até no mercado de telemóveis, fácil, rápida, tem bastantes funções incorporadas para realizar tarefas, e está consistentemente no top 3 das linguagens mais populares do mundo, atualmente sendo a mais popular.

No seu início era apenas utilizada para "Front-end" dos web-sites porém com o aparecimento de "Frameworks" como "node.js"e "npm.js" (que inclusive, estou a utilizar para dar "host" neste web-site) javascript passou a também ser utilizado no "Back-end" dos web-sites

Os pontos fortes desta linguagem são:

  • Versatilidade;
  • Enorme comunidade;
  • Muitas "Frameworks", "Plugins" e projetos "open-source".







HTML

HTML ou "Hypertext Markup Language" (significa "Linguagem de marcação de Hipertexto"), embora não seja considerado uma linguagem é bastante utilizado para a criação da "Front-end" dos websites.

Não é considerado uma linguagem pois não permite adicionar lógica a um código, com a definição de linguagem percebemos que para se enquadrarem nesse tipo, o "candidato" teria de poder dar ordens aos comopnentes físicos, coisa que não acontece com o HTML, pois este apenas serve para criar a página da frente dos websites, não dando dinâmica nenhuma aos mesmos.

Sem a utilização de uma linguagem diferente os sites não teriam dinâmica nenhuma e seriam MUITO aborrecidos, outra "linguagem" que ajuda nisso é "CSS", porém esta também não pode ser considerada uma linguagem como iremos ver a seguir.

CSS

CSS ou "Cascating Style Sheets" é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web, isto é, esta "linguagem" "apenas" serve para colocar o HTML "bonito", embora só faça isso, ela é bastante importante pois sem ela os site seriam muito sem estilo, tudo igual e sem graça, não haveria cor nem formatação apenas texto.

Alguns dos pontos fortes do CSS são:

  • Fácil de aprender;
  • Estruturada de forma em que o codigo é reutilizável;

A programação Web é praticamente composta apenas por CSS, HTML e javascript, sabendo estes três componentes já é possível criar-se um website bastante bonito, estruturado e responsivo.




C++

C++ é uma linguagem orientada e compilada de propósito geral. O seu maior uso é para a criação de video jogos, porém tambem tem outras utilidades como Web Development Server Side, ou seja, criação de código do lado do servidor, e compiladores, editores, ferramentas de programação e programas de redes.

Esta linguagem como ja mencionei foi baseada em C (na verdade, foi tão baseada em C que na sua criação - 1980 - era chamada de "C with classes", tendo depois mudado para C++), e por isso herdou os seus "cons" e os seus "pros" como:

  • Bastantes bibliotecas online para adicionar ao seu código;
  • Bastante rápida;
  • versátil

Alguns dos contras são:

  • Dificil de aprender;
  • Dificil de implementar;
  • Dificil leitura e compreensão do codigo de outra pessoa;
  • A rapidez maior parte das vezes não é necessária, não compensando a dificuldade




Java

Java é uma linguagem de programação orientada a objetos. Diferente das linguagens de programação modernas, que são compiladas para código nativo, Java é compilada para um bytecode que é interpretado por uma máquina virtual (Java Virtual Machine, abreviada JVM). A linguagem de programação Java é a linguagem convencional da Plataforma Java, mas não é a sua única linguagem.

Os maiores pontos positivos de Java são:

  • funciona em qualquer plataforma ou sistema operacional;
  • Grande comunidade (maior facilidade a procurar codigo para o seu projeto);
  • Muitas frameworks;
  • A maquina virtual java suporta muitas outras linguagens;
  • Aloca e liberta automaticamente a memória;

Os maiores pontos negativos são:

  • É lento;
  • Precisa de muita memória;
  • Não tem funcionalidades de backup
  • Código muito abstrato;
